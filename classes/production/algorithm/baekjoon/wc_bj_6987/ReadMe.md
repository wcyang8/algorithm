# BJ 6987 월드컵

### Input

```
5 0 0 3 0 2 2 0 3 0 0 5 4 0 1 1 0 4
4 1 0 3 0 2 4 1 0 1 1 3 0 0 5 1 1 3
5 0 0 4 0 1 2 2 1 2 0 3 1 0 4 0 0 5
5 0 0 3 1 1 2 1 2 2 0 3 0 0 5 1 0 4
```

### Output

```
1 1 0 0
```

### Idea

```
1. 실제 시뮬레이션 돌리기

2. 각 팀끼리 play하여 승무패 3가지 경우 전부 재귀

3. 단, 입력된 결과값보다 승무패가 더 커지면 안된다. (backtracking)
-> 결과값에서 승 무 패를 1씩 빼는 식으로 구현.

4. 기저조건 : 5,6번째 팀까지 play 시켰을 때, 결과값이 모두 0이 되면 true를 리턴한다.
```
