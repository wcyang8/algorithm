# BJ 14725 개미굴

### 문제 이해
```
1. 로봇 개미를 보낸다.

2. 더 내려갈 수 없으면 그 자리에서 움직이지 않고 신호를 보낸다.

3. 내려오면서 알게 된 각 방에 저장된 먹이 정보를 윤수한테 알려줄 수 있다.
```

### Input
```
3           // 먹이 정보 개수
2 B A       // 먹이 정보 K개, 먹이 정보
4 A B C D
2 A C
```

### Output
```
A
--B
----C
------D
--C
B
--A
// 같은 층에 있는 먹이는 사전 순서가 앞서는 먹이부터.
```

### 시간복잡도
```
1 <= N <= 1000
1 <= K <= 15
1 <= t <= 15

트리의 가지를 priority queue를 통해 구현하여보자.
최대 높이는 15
```
