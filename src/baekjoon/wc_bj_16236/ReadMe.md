# BJ 16236 아기상어

### 문제 이해

```
1. N x N 공간에 물고기 M마리, 아기상어 1마리

2. 한 칸에는 물고기 최대 1마리

3. 물고기, 상어는 모두 크기를 가짐. 아기상어 = 2

4. 아기상어는 1초에 상하좌우 1칸 이동

5. 자신의 크기보다 큰 물고기가 있는 칸은 갈 수 없고, 나머지 칸은 모두 이동 가능.

6. 크기가 같은 물고기는 지나갈 수 있으나 먹을 순 없다.

7. 이동 결정 방법
    1. 먹을 수 있는 물고기가 없으면 엄마 상어 도움
    2. 먹을 수 있는 물고기가 1마리라면 그 물고기를 먹으러 감.
    3. 여러 마리면 가장 가까운 물고기
        왼쪽 위부터 먹는다.

8. 이동은 1초

9. 자기 크기만큼 물고기를 먹으면 크기가 1 증가

10. 엄마 상어 도움 없이 몇 초 동안 물고기를 잡아먹을 수 있는지 구해보자.
```

### Input

```
//0: 빈 칸
//1, 2, 3, 4, 5, 6: 칸에 있는 물고기의 크기
//9: 아기 상어의 위치
3
0 0 0
0 0 0
0 9 0

// 2<=N<=20
```

### Output

```
0
```

### idea

```
1. 1초마다 bfs를 사용해서 가장 가까운 고기를 찾는다.

2. 아기상어보다 무게가 큰 물고기는 피해서 간다.

3. 4방 탐색 순서는 위 왼 오 아

4. 위치 이동 후 경험치 확인하여 레벨업 처리

5. bfs를 해도 가까운 고기를 찾지 못하면 끝.
```
